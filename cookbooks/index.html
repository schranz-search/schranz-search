Cookbooks#

This section contain a collection of recipes for other tasks.

Let us know via the issue tracker on our Github Repository. if you have a recipe you would like to see here.

This can go over creating a basic search page in your favorite framework, over modifying default settings of an Index, mapping data to a Model class and over connecting Index updates to your favorite ORM or persistent layer.
